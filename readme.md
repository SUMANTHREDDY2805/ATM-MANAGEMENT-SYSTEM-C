# ATM Management System in C

## Description
This is a simple ATM Management System developed using C programming.

## Features
- Balance Check
- Deposit Money
- Withdraw Money
- Exit Option

## Technologies Used
- C Programming

## How to Run
1. Compile the program:
   gcc atm.c -o atm

2. Run the program:
   ./atm
